# OmniSync

Notification service to keep your devices in sync.
e.g.:

    {
      "registration_id":"416caa38-8539-4f5a-80b6-64444b8481a6",
      "event": {
    	"topic_uri":"levelboy@gmail.com",
        "data":{
          "Type":0
        }
      }
    }
    
Type can have one of these values: 
* 0 = Clippiings 
* 1 = PhoneCall
